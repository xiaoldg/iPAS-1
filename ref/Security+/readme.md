
# Security+

## 課程目標
```
CompTIA Security+已連續3年列為CertCities.com公佈IT業10大熱門證照排行榜的前10名，
是目前國際上最為普及與非廠商導向之資訊安全證照。本課程將學習資訊安全中之基礎技術及核心概念，
讓學員在資安領域下對於廠商的解決方案有更詳盡的認知。課程中協助學員考取CompTIA Security+國際認證。
```

## 課程大綱
```
◆ 安全總論 (Security Overview)
‧CompTIA考試介紹
‧資安趨勢
‧資產之價值與風險管理簡述

◆ 攻擊、惡意軟體與威脅 (Attacks, Malicious software and Threats)
‧常見的網路服務
‧攻擊與惡意軟體介紹
‧威脅說明

◆ 網路架構安全 (Infrastructure Security)
‧儲存媒體安全
‧通訊與網路設備安全
‧網路安全拓樸與架構
‧入侵偵測/防禦系統 (IDS/IPS)
‧蜜罐 (誘捕) 系統
‧事件處理與應變

◆ 弱點與稽核 (Vulnerabilities and Audit)
‧網頁弱點
‧通訊協定弱點
‧檔案傳輸協定弱點
‧無線網路弱點
‧安全基準與強化

◆ 通訊安全與對策 (Communication Security and Countermeasure)
‧安全相關通訊協定介紹
‧遠端存取
‧安全郵件
‧即時通訊
‧網頁連結

◆ 密碼學基本原理 (Basic Cryptography)
‧基本概念
‧加解密說明
‧雜湊演算法
‧加密演算法
‧對稱式 (金鑰) 密碼學

◆ 密碼學的應用 (Cryptographic Applications)
‧非對稱式 (金鑰) 密碼學
‧公開金鑰基礎建設 (PKI) 介紹
‧數位簽章
‧加密應用

◆ 認證、授權與存取控制 (Authentication, Authorization & Access Control)
‧各種身分認證方式介紹
‧授權與責任
‧存取控制說明

◆ 安全政策與管理 (Security Policies and Management)
‧資安政策
‧人力資源
‧風險評鑑與分析
‧風險管理
‧資安相關標準與驗證

◆ 作業安全、營運持續計畫與災難復原計畫 (Operational Security, BCP and DRP)
實體環境安全
資料備份與還原
業務/營運衝擊分析
營運持續計畫(BCP)
災難復原計畫(DRP)

◆ 其他安全相關課題介紹與課程複習 (Other Security related Topics and Course Review)
符合法令與規範
軟體與系統取得
電腦鑑識
課程複習
```
