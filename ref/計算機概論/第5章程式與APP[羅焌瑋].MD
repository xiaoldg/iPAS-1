# 5-1 　程式與 App 
```
● 程式（program）或稱為軟體（software），是由一連串相關指令所組成；
  這些指令是因為共同的目的而組織起來，用來告訴電腦該做什麼，以及該怎麼做。
● 應用程式（application）或簡稱 App，有時也叫做應用軟體（application software），
  是由一組能讓使用者更有生產力，或協助處理個人事務的程式集合而成。
● 作業系統（operating system）是一組用來協調電腦或行動裝置硬體之間所有活動的程式。
 - 工具或公用程式
 - 系統軟體
● 開發人員會選擇用什麼技術來撰寫 App， 
  通常與將來這程式是會被安裝在使用者的裝置上，或透過網路存取使用有關。
 - 原生app
 - 雲端 app
 - Web app
 - 行動裝置Mobile web app
● 程式與 App 的發行途徑有多種形式
 - 零售軟體
 - 客製化軟體
 - 共享軟體
 - 免費軟體
 - 開放原始碼軟體
 - 公共領域軟體
● 授權合約（license agreement），有時稱作 EULA（end-user license agreement），
  是指使用程式或 App 的權利。合約中制訂使用軟體的特定條件，使用者在開始使用軟體前必須接受這些條件。
```
## 5-2 　生產力應用程式
```
● 生產力應用程式（productivity application） 
  是一種協助人們能更有效能和效率，執行在工作場所、學校和家庭中日常活動的軟體。
 - 文書軟體
 - 簡報
 - 試算表
 - 資料庫
 - 筆記
 - 行事曆和聯絡人管理
 - 專案管理
 - 會計
 - 個人財務
 - 法務
 - 報稅
 - 文件轉換
 - 支援服務
 - 企業運算
 
人們使用生產力應用軟體能：
 - 建立
 - 編輯
 - 設定格式
 - 儲存
 - 散布
● 使用者可用文書處理軟體來建立或修改內容。
● 人們常使用文書處理軟體來製作例如信件、報告、郵件標籤和新聞稿等文件。
● 簡報軟體（presentation software）
  是讓使用者為簡報內容建立視覺輔助效果，  而能將想法、訊息和其他資訊傳達給一群人的應用程式。
● 試算表軟體（spreadsheet software）
  是一種能讓使用者將資料組織成直欄和橫列，並對資料進行計算的應用程式。
● 資料庫（database）
  是指將一群資料經過組織，成為方便存取、檢索和使用的資料集合。
    - 資料庫軟體（database software）是讓使用者可以建立、存取和管理資料庫的應用軟體。
● 筆記軟體（note taking software）是一種應用軟體，
  讓使用者可以在頁面上任何位置輸入文字、手寫註解、繪圖、畫草稿、插入相片和連結等，然後將頁面儲存成筆記本的一部分。
● 行事曆和聯絡人管理軟體（calendar and contact management software）
  是一種協助使用者整理行事曆、通訊錄，以及與他人共用這些資料的應用軟體。
● 軟體套件（software suite）是將幾個原本各自獨立的程式包裝成一個單位。
● 專案管理軟體（project management software）讓使用者可用來計畫、排程、追蹤和分析專案中的事件、資源和成本。
● 會計軟體（accounting software）協助公司行號記錄他們的財務交易狀況並整理報告。
● 個人財務軟體（personal finance software） 是一種簡化的會計程式，
  可協助家庭和小型工作室使用者結算支票簿、付款、追蹤個人收入和開支、檢查帳戶餘額、轉帳、追蹤投資，以及評估財務計畫等。
● 法務軟體（legal software）的功用是協助個人、家庭和小型企業準備法律文件，並提供法律資訊。
● 報稅軟體（tax preparation software）是用來引導個人、家庭或小型企業報稅的軟體。
● 文件轉換軟體（document conversion software）可將使用任何應用程式製作的文件，轉存成能保留文件原始外觀的某種共同格式。
● 大型組織或企業因為規模大且分布在不同國家，因此需要特殊的運算解決方案。
 - 人力資源軟體
 - 會計財務軟體
 - 工程或產品研發軟體
 - 製程軟體
 - 行銷軟體
 - 業務軟體
 - 物流軟體
 - 客服軟體
 - 資安軟體
```

## 5-6 　資安工具
```
● 個人防火牆（personal firewall）是一種可偵測並保護個人電腦和它的資料，避免被非法入侵的資安工具。
● 防毒軟體（antivirus program）
  目的是辨識並移除任何在記憶體、儲存媒體或新進檔案中發現的病毒。
● 間諜軟體（spyware）是一種在使用者不知情的情況下，  被放置在電腦或行動裝置中秘密收集使用者資訊，
  之後再將收集到的資訊在使用者上網時傳送到外面去的程式。
● 廣告軟體（adware）
  是一種在網頁、電子郵件或其他 Internet 服務上，以橫幅或彈出視窗顯示網路廣告的程式。
● 間諜軟體移除程式（spyware remover）是一種能偵測並移除間諜軟體和其他類似程式的軟體。
● 廣告軟體移除程式（adware remover）是一種能偵測並移除廣告軟體的程式。
● 垃圾郵件防堵程式
● 網站過濾軟體
● 網路釣魚過濾程式
● 彈出式及隱藏式廣告阻攔程式
```
## 5-7 　 檔案、磁碟和系統管理工具
```
● 檔案管理員（file manager）是一種執行檔案管理相關功能的工具。
● 搜尋工具（search tool），它會根據你輸入的條件，嘗試找出電腦或行動裝置上某個檔案、聯絡人、行事曆、App 等項目的位置。
  搜尋工具會在你的電腦或行動裝置（或加上 Internet）中尋找文件、相片、音樂、聯絡人、行事曆等項目，將搜尋結果整合在一起顯示。
● 看圖軟體（image viewer）是一種能顯示、複製和列印圖像檔案內容的工具。
● 解除安裝程式（uninstaller）是一種可移除程式，同時移除該程式在系統檔案中相關項目的工具。
● 磁碟清理（disk cleanup）工具是負責搜尋並移除不需要的檔案。
● 磁碟重組程式（disk defragmenter）是一種會重新組織電腦硬碟上的檔案和未使用空間， 
  讓作業系統能更快速存取資料，程式執行速度也能加快的工具。
  - 重組動作
● 螢幕保護程式（screen saver）是一種能在滑鼠或鍵盤停止活動一段預定時間後，在裝置的螢幕上自動顯示動態或空白畫面的工具。
● 檔案壓縮工具（file compression tool）能夠縮小檔案大小。
● PC 維護工具（PC maintenance tool）是能找出並修正作業系統問題、偵測和修復硬碟問題，以及改善電腦效能的程式。
● 備份工具（backup tool）能讓使用者複製或備份（back up）已選取檔案或整個儲存媒體的內容到其他儲存位置。
● 還原工具（restore tool）可還原整個過程，並將備份檔回復為原本的格式。
● 電源管理工具（power management tool） 
  用途是監看筆電或行動裝置的電池使用情況、顯示用電量最多的 App，以及預估還能使用多長時間。
```
