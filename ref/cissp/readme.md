# CISSP
### 課程目標：
```
◆ 培養學員具通過CISSP認證考試之實力。

◆  培養學員具備基本通訊、網路安全技術（Firewall, VPN, NAT…等）及系統存取控制等相關概念。

◆  培養學員具評估及建置企業整體資訊安全管理之知識與能力。

◆ 培養學員具資訊安全之整體架構、原理、標準與應用，並具相關之資訊法律、電腦犯罪調查等之知識。
```
### 課程大綱
```
◆  資訊安全趨勢 （Security Trend)

◆  資訊安全與風險管理(Information Security and Risk Management )

◆  存取控制（Access Control)

◆  安全架構與設計(Security Architecture and Design)

◆  作業安全(Operations Security)

◆  通訊與網路安全 （Telecommunications & Network Security ）

◆  密碼學（Cryptography ）

◆ 業務持續運作與災害復原計畫(Business Continuity Planning & Disaster Recovery Planning ）

◆  實體(環境)安全（Physical (Environmental) Security )

◆  應用程式安全(Application Security)

◆  法律、規章、遵循性與調查(Legal, Regulations, Compliance and Investigations)


```
