### bwapp ==> bwapp201811.ova

```
bWAPP 有兩種安裝方式:
[1]單獨安裝，部署到 apache + php + mysql 的環境
[2]安裝虛擬機器版本 bee-box

區別在於虛擬機器版本能夠測試的漏洞更多，比如破殼漏洞，心臟滴血漏洞等在單獨安裝的環境下無法測試。

https://www.freebuf.com/sectool/76885.html
https://sourceforge.net/projects/bwapp/files/bee-box/

1.用 7-zip 解壓
2.用 VMware Workstation 開啟，需要 (5.65 GB (6,075,819,913 位元組) 硬碟空間。
3.預設帳號 bee，預設密碼 bug
```
