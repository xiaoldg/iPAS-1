## 1-1 HTML5基礎格式
```
<!DOCTYPE html>   <!--宣告為HTML5的格式 大小寫沒有關係-->
<html>

<head>          
</head>           

<body>       
</body>

</html>
```


## 1-2.段落與標題
```
<!DOCTYPE html>
<html>
<head> 
<meta charset="utf-8">     <!--編碼用utf-8  中文才不會亂碼-->
<title>學習使我快樂</title>    <!--網頁名稱-->
</head>

<body>         <!--body裡面的會出現在頁面裡面-->
    <h1>这是一个標題</h1>   <!--標題的用法   數字代表大小 1最大  6最小-->  
    <h2>这是一个標題</h2>   <!--標題的用法-->
    <h3>这是一个標題</h3>   <!--標題的用法-->
    <p>我的第一個段落。</p>   <!--段落的用法-->
</body>
</html>
````

## 1-3.連結與圖像
```
<!DOCTYPE html>
<html>
<head> 
<meta charset="utf-8">     <!--編碼用utf-8  中文才不會亂碼-->

</head>

<body>         <!--body裡面的會出現在頁面裡面-->
    <p><a href="http://www.runoob.com">  這是一個連結  </a>   <!--連結的用法--></p>
    <p><img src="C:\Users\PMMD\Desktop\下載.jfif" width="800" height="800" /> <!--圖片的用法--></p>
</body>
</html>
```
