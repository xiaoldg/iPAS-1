# 1_2_資訊安全管理系統


### ISO 27001架構
```
五步驟
1.政策
2.風險分析、評鑑
3.決定管控
4.內部稽核
5.持續改善
```

### PDCA

```
1.Plan(計畫)   規畫 ISMS、
2.Do(實行)     執行ISMS
3.Check(檢查)  監視與審查 ISMS
4.Act (改善)   並採取行動維持與改進 ISMS
```

### 管理階層的支持
```


ISMS 的推動需要組織的人力與財力資源，因此管理階層應藉下列各項支持 ISMS：
建立一份 ISMS 政策。
確保建立 ISMS 各項目標及計畫。
建立資訊安全之各種角色與責任 (R&R)。
向組織傳達推動 ISMS 的重要性。
提供充分資源來推動 ISMS。
決定接受風險的準則。
確保施行內部 ISMS 稽核 (audits)。
施行 ISMS 之管理階層審查 (management review)。

```
